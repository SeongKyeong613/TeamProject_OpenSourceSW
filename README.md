# TeamProject_OpenSourceSW
이 프로젝트는 이미지 파일을 이용해 QR / 바코드를 스캔하고 디코딩하는 도구입니다.
OpenCV를 사용하여 QR코드를 인식합니다.
- Python 3.x
- OpenCV
pip install opencv-python
python scanner.py --image your_image.png
sample_images 폴더에서 테스트 이미지를 확인할 수 있습니다.
